# OPENMIND

### 해당 프로젝트는 코드잇 프론트엔드 스프린트 5기 기초 프로젝트 입니다.
해당 사이트는, 질문을 등록하며 답변을 하는 간단한 사이트입니다.

## 팀장: 임상훈
### 팀원: 우지석, 유호민, 조수연, 홍진호

### 배포 Link : https://codeit-base-project.netlify.app/
